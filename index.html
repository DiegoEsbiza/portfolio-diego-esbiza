Gestor de artículos

Aplicación de escritorio desarrollada en C# sobre .NET Framework, diseñada para gestionar artículos de un catálogo comercial. La app se conecta a una base de datos SQL Server para almacenar y administrar los datos. Entre sus principales funcionalidades se incluyen:
*Visualización y listado de artículos.
*Búsqueda rápida y filtros avanzados por criterios específicos.
*Creación, modificación, eliminación y visualización de detalles de artículos.
